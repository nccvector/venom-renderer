### Features

- [x] Multi-threading
- [x] Bucket Rendering
- [x] wavefront object(.obj) import
- [x] Spatial Subdivision (Octree)
- [x] Lambert BRDF
- [x] Oren nayar BRDF
- [x] Disney BRDF
- [x] Triangles
- [x] Voxels
- [ ] Importance Sampling
- [ ] Miltiple importance sampling
- [ ] Seperate materials for loaded objects
